▸ Meeting Intelligence recordings → compounding intel
Problem

You finish a call, scribble some notes, and move on. A week later you can't remember what was agreed. Every meeting generates context that could make your next meeting better — but it sits in a recording nobody will re-listen to.

What this does

One command after any recorded meeting turns the recording into a structured note — decisions, action items split by owner, competitive mentions, customer pain points. Then updates each attendee's page, catches commitments the first pass missed, and suggests connections to follow up on.

Baseline vs Added

Dex-core: meeting prep briefs and basic note processing.

This adds: deep recording integration, missed action extraction, batch processing, connection suggestions, background queue monitoring.

Commands
/process-meetingsingle recording → structured note
/process-daily-meetingsbatch process today's calls
/find-missed-commitmentsscan for slipped promises
/review-connectionssurface new contacts and links

▸ Self-Improving System every session → smarter next one
Problem

You correct your AI. Next session, it makes the same mistake. Over time, you repeat the same instructions hundreds of times. Your AI never learns your preferences and never gets better at working with you specifically.

What this does

Background automations watch every session and learn. Corrections are captured. Patterns are recorded. A guard runs before every action and checks against known mistakes. It tracks which files you actually use, building a picture of your real workflows over time.

Baseline vs Added

Dex-core: session memory and basic observation.

This adds: automatic learning capture, pattern detection, mistake prevention, usage analysis, a living model of your working style.

Commands
/save-insightcapture compound learnings
/identity-snapshotyour working pattern model
/learningswhat the system has learned
Compound Loops

These workflows form feedback loops — each one makes the others stronger.

  • → Meetings feed relationships feed meetings. Every meeting note updates person pages. Every person page enriches the next prep.
  • → Deals feed planning feed accountability. Deal intel surfaces what needs attention. Planning prioritises it. Cracks detection catches when it slips.
  • → Sessions feed learning feed sessions. Every correction teaches the system. Every pattern becomes a guard rail.
