# DexDiff — See how people actually work with AI

## What this is

DexDiff lets you see how high-performers in your role use AI — then adopt their methodology in one command. No files copied, no code installed. Your AI reads the methodology and generates a version tailored to your setup.

A "diff" describes a job to be done: the problem it solves, how the workflow works, what commands it provides, how it adapts by role. When you adopt a diff, your Dex reads the description, asks you a few questions about your vault, and builds everything fresh for you.

## What's live today (Beta)

**Dave Killeen's profile** — 7 AI workflows built over months as a Field CPO managing 30+ enterprise deals across EMEA:

1. **Meeting Intelligence** — Turn recordings into structured notes, auto-update contact pages, catch missed commitments
2. **Deal Intelligence** — Portfolio health monitoring, transcript-based coaching, continuous account enrichment
3. **Relationship Compounding** — Automatic cross-referencing, context-aware message drafting, stakeholder updates from real activity
4. **Weekly Operating Rhythm** — Deal-first planning, parallel intel pipelines, pillar balance tracking
5. **Accountability & Cracks Detection** — Scan for slipped commitments, project health dashboards, career evidence capture
6. **Thought Leadership Pipeline** — Podcast-to-social, competitive intel scanning, 22-agent content review
7. **Self-Improving System** — Learning capture, pattern detection, mistake prevention, usage analytics

**One-command adoption:** Copy the command from the page, paste in your terminal, guided setup walks you through everything.

**Role adaptation:** Same workflow, different implementation for your role. A sales AE gets deal-stage framing. A PM gets feature request extraction. A CSM gets renewal context.

## How to get started

1. Install Dex — see [heydex.ai](https://heydex.ai)
2. Visit [heydex.ai/diff](https://heydex.ai/diff) (password shared with your invite)
3. Browse Dave's 7 workflows
4. Copy the adopt command for any workflow you like
5. Paste in your terminal — guided setup takes it from there

Or adopt everything at once:
```
dex diff adopt-profile @dave
```
