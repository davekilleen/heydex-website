Interaction Behavior
@dave link:
• Hover: color changes from gray (#888) to white (#f0f0f0)
• Click: navigates to /diff/@dave/
• Provides quick access to "view my profile"

Log out link:
• Hover: color changes from darker gray (#777) to lighter gray (#888)
• Click: clears localStorage auth token and redirects to /diff/
• Subtle styling to not compete with main actions
